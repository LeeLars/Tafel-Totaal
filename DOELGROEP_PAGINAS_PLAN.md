## 1. Doel & Strategie

### Waarom deze pagina's?
- **SEO:** Targeten van specifieke zoekwoorden (bijv. "servies verhuur bruiloft", "tafeldecoratie bedrijfsfeest")
- **Conversie:** Gerichte content die aansluit bij specifieke behoeften per doelgroep
- **Vertrouwen:** Laat zien dat je expertise hebt voor verschillende evenementen
- **Lead Generation:** Verschillende CTA's per doelgroep (offerte aanvraag, direct bestellen)

### Doelgroepen
1. **Particulieren** - Privé feesten, verjaardagen, tuinfeesten
2. **Bedrijven** - Corporate events, netwerkborrels, bedrijfsfeesten
3. **Bruiloften** - Trouwfeesten, recepties, huwelijksdiners
4. **Evenementen** - Festivals, beurzen, grote events

## 2. Pagina Structuur

### 2.1 URL Structuur
```
/particulieren.html
/bedrijven.html
/bruiloften.html
/evenementen.html
```

### 2.2 Template Opbouw (per pagina)

#### Hero Sectie
- **Visueel:** Grote hero afbeelding specifiek voor doelgroep
- **Headline:** Duidelijke waardepropositie (bijv. "Serviesverhuur voor jouw bruiloft")
- **Subtext:** Korte USP (bijv. "Van intieme diners tot grote feesten - zonder afwas")
- **CTA:** Primaire actie (bijv. "Bekijk Pakketten" of "Vraag Offerte Aan")

#### Waarom Tafel Totaal voor [Doelgroep]
- **3-4 USP's** specifiek voor deze doelgroep
- **Iconen + Korte tekst**
- Voorbeelden:
  - Particulieren: "Geen borg vooraf", "Flexibele hoeveelheden", "Bezorging & ophaling"
  - Bedrijven: "Factuur achteraf", "Grote volumes", "Vaste contactpersoon"
  - Bruiloften: "Stijlvolle collecties", "Persoonlijk advies", "Complete styling"

#### Populaire Producten/Pakketten
- **Grid van 3-6 items** die populair zijn bij deze doelgroep
- **Filters:** Bijv. voor bruiloften: "Vintage", "Modern", "Klassiek"
- **CTA:** Link naar product/pakket detail

#### Hoe Het Werkt (Doelgroep-specifiek)
- **4 stappen** aangepast aan deze doelgroep
- Voorbeelden:
  - Particulieren: "1. Kies je servies → 2. Selecteer datum → 3. Bezorging of afhalen → 4. Geniet!"
  - Bedrijven: "1. Offerte aanvragen → 2. Persoonlijk advies → 3. Levering op locatie → 4. Ophaling na event"

#### Inspiratie / Use Cases
- **2-3 voorbeeldscenario's** met afbeeldingen
- Voorbeelden:
  - Particulieren: "Tuinfeest voor 20 personen", "Verjaardagsdiner thuis"
  - Bedrijven: "Netwerkborrel 100+ gasten", "Bedrijfsfeest met buffet"
  - Bruiloften: "Intieme ceremonie 50 gasten", "Groot trouwfeest 150+ gasten"

#### Veelgestelde Vragen (FAQ)
- **5-7 vragen** specifiek voor deze doelgroep
- Accordion/details component
- Voorbeelden:
  - Particulieren: "Kan ik ook kleine hoeveelheden huren?", "Wat als er iets breekt?"
  - Bedrijven: "Kunnen we op factuur betalen?", "Bieden jullie korting bij grote volumes?"

#### CTA Sectie
- **Statement block** met grote call-to-action
- Primaire CTA: "Bekijk Pakketten" of "Vraag Offerte Aan"
- Secundaire CTA: "Neem Contact Op"

## 11. Success Metrics

### KPI's per Pagina
- **Traffic:** Organische bezoekers per maand
- **Engagement:** Tijd op pagina, scroll depth
- **Conversie:** Click-through naar producten/pakketten
- **Leads:** Offerte aanvragen via deze pagina's
- **SEO:** Ranking voor target keywords

### Targets (na 3 maanden)
- 500+ organische bezoekers per maand (totaal alle 4 pagina's)
- 3+ minuten gemiddelde tijd op pagina
- 15%+ click-through naar producten
- 5%+ conversie naar offerte/contact
